E-Commerce Inventory & Order Management System
A full-stack web application for managing products, inventory, and orders with role-based access for Admins and Customers.

Built using React, Node.js, Express, Sequelize, and MySQL.
Features
Authentication
- JWT-based authentication
- Role-based access control (Admin / Customer)
Admin (Product Manager)
- Add, edit, delete products
- Upload and manage product images
- Inventory management with low-stock warnings
- Dashboard with analytics (charts)
- View all customer orders
- Dispatch or cancel pending orders
- View customer ID and product images per order
Customer
- Browse products with images
- Search, filter, and sort products
- Add items to cart with quantity limits
- Persistent cart using localStorage
- Place orders
- View order history with product details
Tech Stack
Frontend:
- React
- React Router
- Chart.js
- HTML, CSS, JavaScript

Backend:
- Node.js
- Express.js
- Sequelize ORM
- MySQL
- Multer
- JWT Authentication
